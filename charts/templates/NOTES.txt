This Helm chart is a zabbix monitoring system for SPC Infrastructure Resources.

```bash
** Helm Repository : spc-chart

$ helm install {{ .Release.Name }} spc-chart/"spc-mariadb-zabbix" -n {{ .Release.namespace}}                  # Helm 3
$ helm install --name {{ .Release.Name }} spc-chart/"spc-mariadb-zabbix" -n {{ .Release.namespace}}           # Helm 2
```

Please be patient while the chart is being deployed

Tip:

  Watch the deployment status using the command: kubectl get pods -w --namespace {{ .Release.Namespace }}

Database Root Password:

  Username: root
  Password : $(kubectl get secret --namespace {{ .Release.Namespace }} {{ .Release.Name }}-secret -o jsonpath="{.data.mariadb-root-password}" | base64 --decode)

To connect to your database:

  1. Run a pod that you can use as a client:

      kubectl run {{ .Release.Name }}-dbclient --rm --tty -i --restart='Never' --image  {{ template "mariadb.client.image" . }} --namespace {{ .Release.Namespace }} --command -- bash

  2. To connect to mariadb service (read/write):

      mysql -h {{ .Release.Name }}-db.{{ .Release.Namespace }}.svc.cluster.local -uroot -p {{ .Values.service.mariadb.port }}


To upgrade this helm chart:

[example]
Obtain the password as described on the 'Administrator credentials' section and set the 'database.root.password' parameter as shown below:

```bash
ROOT_PASSWORD="Some difference password"
helm upgrade {{ .Release.Name }} spc-chart/"spc-mariadb-zabbix" --set database.root.password=$ROOT_PASSWORD
```
